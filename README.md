# WorkTravelBal

A modern web application for work and travel balance management.

## Getting Started

This project is set up as a Node.js application. To get started:

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

## Project Structure

```
worktravelbal/

```
